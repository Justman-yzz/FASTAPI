### 3) Black / Ruff

**black**: 코드 모양을 자동으로 통일해주는 “줄맞춤 로봇(포매터)”

**ruff**: 코드에서 실수/나쁜 습관을 찾아주는 “검사기(린터)”

- 설치

poetry add --group=dev black==24.10.0 ruff==0.8.2

![image.png](attachment:57a82d39-cd8e-46c2-8d8b-845adec17f46:image.png)

실수로 위의 내용을 그대로 아래에 두번 복사 붙여놓기

### 결과

- 최종적으로 `all checks passed` 확인

### 4) Mypy

mypy는 파이썬 코드를 **실행하기 전에** “타입(자료형) 관련 실수”를 찾아주는 도구

즉, 런타임 에러가 나기 전에 미리 잡아주는 “사전 검사기”

- 설치

poetry add --group=dev mypy==1.13.0

![image.png](attachment:d356934d-4d43-45b9-8fc7-221c60a91977:image.png)

- pyproject.html(설정)에 추가

[tool.mypy]
python_version = 3.13
strict = true

![image.png](attachment:6419c66a-4e48-457b-941e-dfdf5561017b:image.png)

### 5) Pytest

pytest는 파이썬 코드가 제대로 동작하는지 확인하는 **테스트 실행 도구**

나중에 코드를 수정해도 망가졌는지 바로 확인

- 설치

poetry add --group=dev pytest
